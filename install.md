# 環境要求

## 前端

- [nodejs](https://nodejs.dev/)
- [yarn](https://yarnpkg.com/lang/en/docs/install/)
## 後端

- [golang](https://golang.org/dl/)
- [gcc](https://gcc.gnu.org/)

# 步驟

## 安裝yarn
```bash
    cd fe
    npm install --global  yarn
```

## 安裝node module
```bash
    yarn install
```

## 啟動前端
```bash
    npm run dev
    或
    npm run serve
```

## 編譯並啟動後端
```bash
    cd be
    go run ...
```
